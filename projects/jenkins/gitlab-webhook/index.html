GitLab Webhook

• server

Jenkins triggered automatically via GitLab webhook on every push or MR. Configured GitLab plugin, auth tokens and branch-specific pipeline rules.

Setup Steps
  1. Install GitLab Plugin in Jenkins → Manage Plugins
  2. Create an API token in Jenkins for GitLab auth
  3. GitLab: Settings → Webhooks → Add Jenkins URL
  4. Set trigger events: Push + Merge Requests
  5. Test the webhook and verify the build fires
webhook-test.sh bash
# Jenkins URL to set in GitLab Webhook: JENKINS_URL="http://jenkins:8080" JENKINS_USER="gitlab-trigger" JENKINS_TOKEN="your-api-token" # Test manually: curl -X POST "${JENKINS_URL}/job/my-pipeline/build" \ --user "${JENKINS_USER}:${JENKINS_TOKEN}"
