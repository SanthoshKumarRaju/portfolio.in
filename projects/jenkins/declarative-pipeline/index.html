Declarative Pipeline

• localhost

Full declarative Jenkinsfile pipeline with stages: Checkout → Build → Test → Docker Build → Deploy with Blue Ocean dashboard.

Setup Steps
  1. Create a Jenkinsfile in the root of your Git repo
  2. Jenkins: New Item → Pipeline → Pipeline from SCM
  3. Point to your repo and Jenkinsfile path
  4. Trigger a build and watch Blue Ocean
Jenkinsfile groovy
pipeline { agent any environment { IMAGE = "myapp:${BUILD_NUMBER}" } stages { stage('Checkout') { steps { checkout scm } } stage('Build') { steps { sh 'npm ci' } } stage('Test') { steps { sh 'npm test' } } stage('Docker') { steps { sh "docker build -t ${IMAGE} ." } } stage('Deploy') { steps { sh "docker service update --image ${IMAGE} app_web" } } } }
