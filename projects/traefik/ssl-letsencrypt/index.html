// Traefik

SSL + Let's Encrypt

• server

Traefik as reverse proxy for all Docker services with automatic HTTPS via Let's Encrypt ACME. Docker label-based routing — no per-service config files.

Setup Steps
  1. touch acme.json && chmod 600 acme.json
  2. Write traefik.yml with entrypoints and certificatesResolvers
  3. Add Traefik labels to each service in compose file
  4. Point domain DNS A-record to server IP
  5. Deploy: docker compose up -d — certs auto-requested
docker-compose.yml yaml
services: traefik: image: traefik:v3.0 ports: ["80:80", "443:443"] volumes: - /var/run/docker.sock:/var/run/docker.sock - ./acme.json:/acme.json - ./traefik.yml:/traefik.yml myapp: image: myapp:latest labels: - traefik.enable=true - traefik.http.routers.app.rule=Host(`app.domain.com`) - traefik.http.routers.app.tls.certresolver=letsencrypt - traefik.http.services.app.loadbalancer.server.port=3000
