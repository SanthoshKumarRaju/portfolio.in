// AWS

S3 Static Website

• free tier

Static site hosted on S3 with public bucket policy, versioning, CloudFront CDN for global caching and HTTPS — entirely within AWS free tier.

Setup Steps
  1. Create S3 bucket matching your domain name
  2. Enable Static Website Hosting in bucket Properties
  3. Add public GetObject bucket policy
  4. Upload: aws s3 sync ./dist s3://my-bucket/ --delete
  5. Create CloudFront distribution → S3 endpoint
  6. Set error page to index.html for SPA routing
  7. Create Route 53 A-record alias → CloudFront
deploy.sh bash
aws s3 mb s3://my-site --region ap-south-1 aws s3 website s3://my-site/ \ --index-document index.html \ --error-document index.html aws s3 sync ./dist s3://my-site/ \ --delete --cache-control "max-age=31536000" aws cloudfront create-invalidation \ --distribution-id EXXXXX --paths "/*"
