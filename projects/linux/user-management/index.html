User Management

• server

User and group administration on a Linux server — service accounts, sudo rules, SSH key-based auth and file permission auditing.

Setup Steps
  1. Create service user: useradd -m -s /bin/bash deploy
  2. Add to sudo: usermod -aG sudo deploy
  3. Create .ssh/authorized_keys with your public key
  4. Verify with: su - deploy
