// Docker Swarm

Single Node Swarm

• localhost

Initialised Docker Swarm on a single machine. Deployed monitoring stack (Grafana, Prometheus) as Swarm services with rolling update policy.

Setup Steps
  1. Initialise swarm: docker swarm init
  2. Create overlay network: docker network create --driver overlay mon-net
  3. Write docker-stack.yml for monitoring
  4. Deploy: docker stack deploy -c docker-stack.yml monitoring
  5. Verify: docker service ls
docker-stack.yml yaml
version: '3.8' services: prometheus: image: prom/prometheus:latest ports: ["9090:9090"] deploy: {replicas: 1} grafana: image: grafana/grafana:latest ports: ["3000:3000"] deploy: {replicas: 1} networks: mon-net: {driver: overlay}
