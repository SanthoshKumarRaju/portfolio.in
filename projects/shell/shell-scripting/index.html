Shell Scripting

• localhost

Bash automation scripts for sysadmin tasks — backup rotation, log archiving, health check pings and cron scheduling.

Setup Steps
  1. Create a scripts/ directory in your home folder
  2. Write backup.sh that tarballs /var/app and uploads to S3
  3. Add execute permission: chmod +x backup.sh
  4. Schedule with crontab -e for nightly runs at 2 AM
backup.sh bash
#!/bin/bash DATE=$(date +%Y%m%d-%H%M) BACKUP="/tmp/backup-${DATE}.tar.gz" tar -czf $BACKUP /var/app aws s3 cp $BACKUP s3://my-backups/ rm -f $BACKUP echo "✓ Backup ${DATE} uploaded"
