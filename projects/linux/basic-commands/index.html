Basic Commands

• localhost

Core Linux command-line operations on Ubuntu VM. File navigation, permissions, process management, apt package manager and systemd service control.

Setup Steps
  1. Install Ubuntu 22.04 on VirtualBox or use WSL2 on Windows
  2. Run sudo apt update && sudo apt upgrade -y
  3. Practise commands below in a safe /tmp/lab directory
  4. Use journalctl -u nginx -f to follow service logs
commands.sh bash
pwd # print working directory ls -lah # long list + hidden files find / -name "*.conf" 2>/dev/null chmod 755 script.sh # rwxr-xr-x chown deploy:deploy /var/app ps aux | grep nginx systemctl status nginx journalctl -u nginx -f df -h && free -m
