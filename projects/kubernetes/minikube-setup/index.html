Minikube Setup

• localhost

Local Kubernetes cluster with Minikube. Deployed multi-container app using Deployments, Services, ConfigMaps, Secrets and Ingress controller.

Setup Steps
  1. Install Minikube from storage.googleapis.com
  2. Start: minikube start --driver=docker --cpus=2 --memory=4096
  3. Enable ingress: minikube addons enable ingress
  4. Apply manifests: kubectl apply -f k8s/
  5. Access: minikube service app-service
deployment.yaml yaml
apiVersion: apps/v1 kind: Deployment metadata: {name: myapp} spec: replicas: 3 selector: {matchLabels: {app: myapp}} template: metadata: {labels: {app: myapp}} spec: containers: - name: myapp image: myapp:latest ports: [{containerPort: 3000}] resources: limits: {cpu: "500m", memory: "256Mi"}
