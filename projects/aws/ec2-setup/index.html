EC2 Setup

• free tier

Launched and configured t2.micro EC2 — Security Groups, Elastic IP, SSH key pair, user-data bootstrap and IAM instance profile.

Setup Steps
  1. Launch EC2 t2.micro (Ubuntu 22.04 AMI)
  2. Create Security Group: allow SSH (22), HTTP (80), HTTPS (443)
  3. Allocate and attach Elastic IP
  4. SSH: ssh -i key.pem ubuntu@<elastic-ip>
  5. Run user-data bootstrap to install Docker and Nginx
user-data.sh bash
#!/bin/bash apt update -y && apt upgrade -y curl -fsSL https://get.docker.com | sh usermod -aG docker ubuntu systemctl enable --now docker apt install -y nginx awscli systemctl enable --now nginx
