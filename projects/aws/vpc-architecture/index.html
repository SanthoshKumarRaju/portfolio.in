// AWS

VPC Architecture

• server

Custom multi-tier VPC: public subnets for ALB and NAT Gateway, private subnets for application servers and RDS. Bastion host for secure SSH.

Setup Steps
  1. Create VPC: CIDR 10.0.0.0/16
  2. Create 2 public subnets (10.0.1.0/24, 10.0.2.0/24) across AZs
  3. Create 2 private subnets (10.0.10.0/24, 10.0.11.0/24)
  4. Create and attach Internet Gateway
  5. Create NAT Gateway in public subnet with Elastic IP
  6. Route tables: public→IGW, private→NAT GW
  7. Security Groups per tier (web/app/db)
