// Grafana

Prometheus + Grafana

• localhost

Full observability stack: Prometheus scrapes node-exporter, Grafana shows custom dashboards for CPU, memory, disk, network and HTTP request rates.

Setup Steps
  1. Run node-exporter on each host
  2. Write prometheus.yml with scrape targets
  3. Deploy stack with Docker Compose
  4. Open Grafana: http://localhost:3000 (admin/admin)
  5. Add Prometheus data source → Import dashboard ID 1860
  6. Create custom Alertmanager alert rules
prometheus.yml yaml
global: scrape_interval: 15s scrape_configs: - job_name: 'prometheus' static_configs: - targets: ['localhost:9090'] - job_name: 'node-exporter' static_configs: - targets: ['node-exporter:9100'] - job_name: 'app' static_configs: - targets: ['app:3000']
