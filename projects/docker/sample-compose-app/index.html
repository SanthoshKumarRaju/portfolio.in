// Docker

Sample Compose App

• localhost

Full-stack multi-container app: React frontend + Node.js API + PostgreSQL + Nginx. Demonstrates healthchecks, depends_on ordering, named volumes and env injection.

Setup Steps
  1. cd into the project directory
  2. Copy .env.example to .env and fill values
  3. Run docker compose up --build -d
  4. Frontend: http://localhost:3000   API: http://localhost:4000/api
  5. Logs: docker compose logs -f api
docker-compose.yml yaml
services: db: image: postgres:15-alpine environment: POSTGRES_DB: "${DB_NAME}" POSTGRES_USER: "${DB_USER}" POSTGRES_PASSWORD: "${DB_PASS}" healthcheck: test: ["CMD-SHELL", "pg_isready -U ${DB_USER}"] api: build: ./api depends_on: {db: {condition: service_healthy}} ports: ["4000:4000"] frontend: build: ./frontend ports: ["3000:80"]
