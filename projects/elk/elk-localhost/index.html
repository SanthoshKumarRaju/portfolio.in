// ELK Stack

ELK on Localhost

• localhost

Full ELK Stack via Docker Compose — Elasticsearch single-node, Logstash pipeline, Kibana. Built from scratch and used in production at Solutyics.

Setup Steps
  1. sudo sysctl -w vm.max_map_count=262144
  2. Write docker-compose.yml with ES, Logstash, Kibana
  3. Create logstash pipeline config
  4. Start: docker compose up -d
  5. Open Kibana at http://localhost:5601
  6. Create Index Pattern → Discover → Dashboard
docker-compose.yml yaml
services: elasticsearch: image: elasticsearch:8.11.0 environment: - discovery.type=single-node - xpack.security.enabled=false - ES_JAVA_OPTS=-Xms512m -Xmx512m ports: ["9200:9200"] logstash: image: logstash:8.11.0 volumes: [./pipeline:/usr/share/logstash/pipeline] ports: ["5044:5044"] kibana: image: kibana:8.11.0 ports: ["5601:5601"] depends_on: [elasticsearch]
