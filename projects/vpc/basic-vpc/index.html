// VPC & Networking

Basic VPC Setup

• server

Custom multi-tier AWS VPC from scratch — IGW, route tables, NAT Gateway, Security Groups for web/app/db tiers and NACLs.

Setup Steps
  1. Create VPC with CIDR 10.0.0.0/16
  2. Create public subnets across 2 AZs — enable auto-assign public IP
  3. Create private subnets in same AZs
  4. Create and attach Internet Gateway
  5. Allocate Elastic IP → Create NAT Gateway in public subnet
  6. Route tables: public→IGW, private→NAT
  7. Create Security Groups per tier
sg-rules.sh bash
# Web tier SG — public facing aws ec2 authorize-security-group-ingress \ --group-id sg-web --protocol tcp \ --port 443 --cidr 0.0.0.0/0 # App tier — from web SG only aws ec2 authorize-security-group-ingress \ --group-id sg-app --protocol tcp \ --port 3000 --source-group sg-web # DB tier — from app SG only aws ec2 authorize-security-group-ingress \ --group-id sg-db --protocol tcp \ --port 5432 --source-group sg-app
